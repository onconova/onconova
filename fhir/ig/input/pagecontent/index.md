# Onconova FHIR Implementation Guide

Welcome to the official FHIR Implementation Guide for **Onconova**.

This guide provides a comprehensive overview of the FHIR-based interface to Onconova, supporting interoperability, data exchange, and integration with clinical and research systems. It complements the OpenAPI specification and is designed for healthcare organizations, developers, and integrators seeking to leverage Onconova's oncology data and workflows.

## About Onconova
Onconova is a precision oncology platform focused on enabling advanced cancer research, clinical decision support, and data-driven care. The FHIR interface exposes key resources, profiles, and value sets to facilitate standardized data exchange and integration with EHRs, registries, and analytics platforms.
