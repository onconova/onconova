Resource Profile: Venous Invasion Category

Official URL: http://luisfabib.github.io/onconova/StructureDefinition/onconova-venous-invasion-category Version: 0.1.0
Active as of 2025-10-09 Computable Name: OnconovaTNMVenousInvasionCategory

A profile representing the venous invasion category for a cancer patient.

This profile extends the base mCODE TNMCategory profile to specify the new TNM category.

Usages:

  • Refer to this Profile: TNM Stage Group
