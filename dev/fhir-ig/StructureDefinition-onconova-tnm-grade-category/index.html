    Resource Profile: TNM Grade Category¶
    Official URL:http://luisfabib.github.io/onconova/StructureDefinition/onconova-tnm-grade-category Version:0.1.0
    Active as of 2025-10-15 Computable Name:OnconovaTNMGradeCategory

    A profile representing the TNM grade category for a cancer patient. This profile extends the base mCODETNMCategory profileto specify the new TNM category.

    Usages:

    • Refer to this Profile: TNM Stage Group
    • CapabilityStatements using this Profile: Onconova FHIR REST Capability Statement

    You can also check for usages in the FHIR IG Statistics
