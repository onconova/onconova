    Logical Definition (CLD)¶

    Expansion¶

    Expansion from tx.fhir.org based on SNOMED CT International edition 01-Feb 2025

    This value set contains 301 concepts

    Explanation of the columns that may appear on this page:

    Level A few code lists that FHIR defines are hierarchical - each code is assigned a level. In this scheme, some codes are under other codes, and imply that the code they are under also applies
    System The source of the definition of the code (when the value set draws in codes defined elsewhere)
    Code The code (used as the code in the resource instance)
    Display The display (used in thedisplayelement of aCoding). If there is no display, implementers should not simply display the code, but map the concept into their application
    Definition An explanation of the meaning of the concept
    Comments Additional notes about how to use the code
